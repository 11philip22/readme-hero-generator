# Hero Generator

Generate README hero banners as PNG images.

## What It Does

- Renders a banner preview in the browser.
- Lets you edit:
  - project name
  - subtitle
  - description
  - tags
  - accent color
  - background style (`NONE`, `DOTS`, `WAVE`, `VOID`, `ZEBRA`)
  - light/dark mode
- Exports a PNG with a 1400x280 design size (`@2x` render for sharper output).

## Run Locally

```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
npm run preview
```
