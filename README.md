## What It Does

- Renders a banner preview in the browser.
- Lets you edit:
  - project name
  - subtitle
  - description
  - tags
  - accent color
  - background style (`NONE`, `DOTS`, `GRID`, `SCAN`, `WAVE`, `VOID`, `ZEBRA`)
  - light/dark mode
- Exports a PNG with a 1400x280 design size (`@2x` render for sharper output).
